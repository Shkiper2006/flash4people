# Flash4People

Браузерный мессенджер с комнатами, текстовыми и голосовыми чатами, хранением истории и файлов в `DB.dat`.

## Возможности

- Регистрация и вход.
- Комнаты с историей сообщений.
- Онлайн/оффлайн статусы пользователей.
- Приглашения в комнату с автоматическим таймаутом 5 минут.
- Текстовые сообщения, файлы и изображения.
- Эмодзи для чата.
- Голосовой чат: авто-включение микрофона, выбор устройства, демонстрация экрана, индикатор активности микрофона.
